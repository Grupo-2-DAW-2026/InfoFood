# InfoFood - Trazabilidad Alimentaria

Proyecto de gestión de productos con trazabilidad completa, información nutricional y alérgenos.

## Requisitos para el maestro:
1. Clonar el repositorio.
2. Ejecutar `composer install`.
3. Configurar la BD.
4. Importar el archivo `/database/sql/el_pozo_db.sql` para tener los datos de prueba.
5. Ejecutar `php artisan serve`.
